# 💰 Expense Tracker

An expense management web application that helps users track income, expenses, and overall financial activity through an intuitive dashboard.

---

## 📖 Project Overview

This application enables users to record daily transactions, categorize expenses, and monitor spending patterns to improve personal financial management.

---

## ✨ Features

* Add income and expenses
* Edit and delete transactions
* Expense categorization
* Balance calculation
* Transaction history
* Responsive user interface
